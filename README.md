# ModelExplainer

SHAP and LIME for explaining black-box ML models.

Result: Identified capital-gain as the most important feature.

Run:
git clone https://github.com/tinkywinkygluestick799-dotcom/modelexplainer
cd modelexplainer
pip install -r requirements.txt
python explain_model.py

Dependencies: Python 3.8+, SHAP, LIME, scikit-learn, XGBoost
